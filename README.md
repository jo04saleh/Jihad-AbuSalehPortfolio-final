## ✨ About

Portfolio شخصي مبني بـ React + Vite يعرض مشاريعي ومهاراتي كـ Full-Stack Developer. صُمِّم بتفاصيل دقيقة لتقديم تجربة مستخدم سينمائية وانسيابية.

---

## 🎨 Features

- **Particle Background** — شبكة جسيمات متحركة تفاعلية
- **Custom Cursor** — cursor مخصص بتأثير lerp ناعم
- **3D Project Carousel** — عرض المشاريع بتأثير ثلاثي الأبعاد مع swipe
- **Ambient Sound System** — صوت فضائي محيطي مبني بـ Web Audio API
- **UI Click Sounds** — أصوات تفاعلية على كل زر
- **Animated Skills** — أشرطة مهارات تتحرك عند الظهور
- **Dark / Light Mode** — دعم كامل للوضعين
- **Fully Responsive** — متجاوب مع جميع الشاشات والأجهزة
- **Mobile Navigation** — hamburger menu للجوال
- **Scroll Animations** — انيميشن عند التمرير بـ Framer Motion

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/jo04saleh/Jihad-AbuSalehPortfolio-final.git

# Navigate to project
cd Jihad-AbuSalehPortfolio-final

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
